# Econometric Analysis of Wage Determinants

## Описание проекта

В данном проекте проводится эконометрический анализ факторов, влияющих на уровень заработной платы, на основе микроданных Российского мониторинга экономического положения и здоровья населения (RLMS).

Цель исследования — оценить влияние характеристик человеческого капитала и демографических факторов на доход работников с использованием методов эконометрики.

Основное внимание уделяется таким переменным, как образование, трудовой опыт и возраст, которые согласно экономической теории являются ключевыми детерминантами заработной платы.

## Используемые данные

В работе используется микроданные RLMS — одного из крупнейших панельных обследований домохозяйств в России.

В проекте были выполнены:

- очистка и подготовка данных

- формирование аналитической выборки

- создание новых переменных для моделирования

## Методы анализа

Для исследования использовались следующие методы анализа данных и эконометрики:

- Exploratory Data Analysis (EDA)

- анализ распределений и выбросов

- корреляционный анализ

- линейная регрессия (OLS)

- логарифмическая модель заработной платы

- диагностические тесты эконометрической модели:

- Breusch–Pagan test (гетероскедастичность)

- VIF (мультиколлинеарность)

- RESET test (проверка спецификации модели)

- Также в проекте используется PCA (Principal Component Analysis) для анализа структуры признаков.

## Основные результаты

Анализ показал устойчивые зависимости между характеристиками человеческого капитала и уровнем заработной платы:

- образование оказывает положительное влияние на доход

- трудовой опыт увеличивает заработную плату, но с убывающей отдачей

- демографические характеристики также оказывают влияние на уровень дохода

Полученные результаты согласуются с теорией человеческого капитала и эмпирическими исследованиями рынка труда.
